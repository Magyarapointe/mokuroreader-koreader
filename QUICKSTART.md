## Common Questions

**Q: How do I process manga with mokuro?**
```bash
pip install mokuro
mokuro path/to/manga.cbz
```

**Q: Plugin doesn't appear?**
- Check folder is named `mokuroreader.koplugin` (exactly)
- Verify it's in the right plugins folder
- Restart KOReader completely

**Q: Selection not working?**
- **Hold** (don't tap) on text
- Hold for at least 0.5 seconds
- Make sure to hold on the text area

**Q: Dictionary not opening?**
- Install a Japanese dictionary in KOReader settings
- Ensure dictionary is enabled
